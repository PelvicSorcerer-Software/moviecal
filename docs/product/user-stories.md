User stories

- As a user, I can search for movies so I can find titles to track.
- As a user, I can add a movie to my watchlist so I remember to watch it.
- As a user, I can remove a movie from my watchlist.
- As a user, I can create a private calendar subscription URL so my iOS Calendar shows my watchlist release dates.
- As a user, I can regenerate my calendar token if it is compromised.

Each story should include acceptance criteria and automated tests where sensible.
