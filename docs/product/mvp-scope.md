MVP scope

In-scope for MVP:
- Movie search using TMDb
- Per-user watchlist (add/remove)
- Generate private iCalendar feed URL (unguessable token)
- Feed includes events for confirmed release dates and updates when dates change
- Basic UI for search, watchlist, and feed management

Out-of-scope for MVP (see non-goals)
- Social sharing, public watchlists, or full mobile apps
- Advanced recommendations or analytics
